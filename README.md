# Product Billing Application
# Overview 
It allows users to add products to a basket, calculates the subtotal, applies special offers, and displays the final total with savings.

# Features 
Product Listing: Displays products with name, price, and image.

Add to Cart: Users can add products to the basket.

# Cart Summary: 

Shows products added with quantity and line totals.

Displays subtotal before offers.

Lists special offers applied with individual savings.

Shows total savings and final total.

# Special Offers Implemented: 

Buy one Cheese, get one Cheese free.

Buy Soup, get half-price Bread.

Get one-third off Butter.

# Responsive Design:
Mobile-friendly layout with CSS media queries.

# Tech Stack
React (UI library)

Redux Toolkit (state management)

TypeScript (type safety)

# Installation & Setup
1. Clone the repository:

git clone https://github.com/SudhaMandave/Product-Billing-App.git
cd Product-Billing-App

2. Install dependencies:
npm install

3. Start the development server:
npm run dev   # if using Vite

npm start     # if using CRA
